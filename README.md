## FanIDE

Netbeans plugin to support the Fantom/Fanx Language.

It forked from tcolar's FantomIDE (http://fantomide.colar.net/)

### Features
- Syntax Highlighting
- Semantic Error
- Completion with Doc
- Go to declaration
- Debugging
- Outline view
